# Observability Project (FastAPI + Prometheus + Grafana)

## 🚀 Overview
This project demonstrates monitoring and observability using FastAPI, Prometheus, and Grafana.

## 🛠 Tech Stack
- FastAPI (Python)
- Prometheus (Metrics collection)
- Grafana (Visualization)
- Docker & Docker Compose

## 📊 Features
- API request count tracking
- API latency monitoring
- Custom metrics exposure via `/metrics`
- Grafana dashboard visualization

## ▶️ How to Run

```bash
docker-compose up --build
